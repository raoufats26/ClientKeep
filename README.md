# ClientKeep — Setup Guide

## Phase 1: Project Setup + Auth + Core Features

---

## 🐍 Step 1 — Create Virtual Environment

```bash
# In your clientkeep/ folder
python -m venv venv

# Activate it:
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate
```

---

## 📦 Step 2 — Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🐘 Step 3 — Create PostgreSQL Database

Open pgAdmin or psql and run:

```sql
CREATE DATABASE clientkeep;
```

Then update your `.env` file with your PostgreSQL credentials:

```
DATABASE_URL=postgresql://postgres:YOUR_PASSWORD@localhost:5432/clientkeep
SECRET_KEY=any-long-random-string-here
```

---

## 🗄️ Step 4 — Run Database Migrations

```bash
flask db init
flask db migrate -m "Initial models"
flask db upgrade
```

---

## 🚀 Step 5 — Run the App

```bash
flask run
```

Open: http://127.0.0.1:5000

## 🛟 Common Issues

**Problem:** `ModuleNotFoundError: No module named 'psycopg2'`
**Fix:** `pip install psycopg2-binary`

**Problem:** `sqlalchemy.exc.OperationalError` (can't connect to DB)
**Fix:** Make sure PostgreSQL is running and your `.env` DATABASE_URL is correct

**Problem:** Flask says "No application found"
**Fix:** Make sure you ran `flask run` with your venv activated and you're in the `clientkeep/` folder

**Problem:** Migrations folder already exists error
**Fix:** Delete the `migrations/` folder and re-run `flask db init`
